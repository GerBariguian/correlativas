# Correlativas

Asistente académico para saber qué cursar, qué rendir y qué materias te están frenando.

## Stack

- React
- Vite
- LocalStorage
- CSS puro
- lucide-react

## Cómo correrlo

```bash
npm install
npm run dev
```

Después abrí la URL que te muestre Vite.

## Primer MVP

- Dashboard de progreso
- Estados de materias
- Podés cursar
- Podés rendir final
- Bloqueadas
- Desbloqueos
- Asesor académico básico
